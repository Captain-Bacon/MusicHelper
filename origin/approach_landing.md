
# Approach: Landing Progressions Reference

Below is a list of common landing (resolution) chord progressions, each labeled with their harmonic outline, common voicing, and the tone or "character" they evoke:

| Progression | Example Chords         | Voicing/Style        | Character / Usage          |
|-------------|------------------------|----------------------|----------------------------|
| **V → I**   | G → C                  | Triads               | Straight, clean            |
| **bVII → I**| Bb → C                 | Triads               | Rocky, anthemic            |
| **bII → I** (tritone sub) | Db → C   | Substitutions        | Jazzy                      |
| **IV → V → I** | F → G → C           | Triads               | Straight, clean            |
| **ii → V → I** | Dm → G → C          | Triads               | Warm, smooth               |
| **bVI → V → I** | Ab → G → C         | Triads               | Dramatic, cinematic        |
| **IV → iv → I** | F → Fm → C         | 7ths throughout      | Dramatic, cinematic        |
| **ii → V7 → I** | Dm → G7 → C        | Dom. 7th on V        | Warm, smooth               |
| **bVI → V7 → I** | Ab → G7 → C       | Dom. 7th on V        | Dramatic, cinematic        |
| **IV → V7 → I** | F → G7 → C         | Dom. 7th on V        | Straight, clean            |
| **ii7 → V7 → IMaj7** | Dm7 → G7 → CMaj7 | 7ths throughout   | Jazzy                      |
| **bVII → IV → V → I** | Bb → F → G → C | Triads             | Rocky, anthemic            |
| **#IVdim7 → V7 → I** | F#dim7 → G7 → C  | Substitutions      | Gospel flourish            |
| **iii → vi → ii → V → I** | Em → Am → Dm → G → C | Triads  | Warm, smooth     |

---

- **Voicing** describes the chord type/texture commonly used (Triads, 7ths, etc).
- **Character** gives a sense of the stylistic "flavor" or common genre/feel.

*Use this table as a quick cheat sheet for resolving progressions in songwriting or analysis!*
